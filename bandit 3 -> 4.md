# Bandit level 3 -> 4

## Objective  
To read the hidden file inside the inhere directory.

## Commands Used  
ls -> to list files in the directory  
cd inhere -> to enter the inhere directory  
ls -a -> to show hidden files  
cat -> to read the file  

## Explanation  
First, ls was used to see the inhere directory.
Then cd was used to move into it.
ls -a was used to list hidden files.
Finally, cat was used to read the hidden file.

## Outcomes  
Learned how to view and read hidden files in Linux.
