# Bandit level 8 -> 9

## Objective  
To find the password in data.txt that appears only once.

## Commands Used  
sort data.txt | uniq -u

## Explanation  
The sort command was used to arrange the data.
The output was passed to uniq -u to show the line that appears only one time.
This line contains the password.

## Outcomes  
Learned how to sort data and find unique entries in a file.
