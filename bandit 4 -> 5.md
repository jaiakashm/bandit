# Bandit level 4 -> 5

## Objective  
To read the password from the human-readable file.

## Commands Used  
ls -> to list directories  
cd inhere -> to enter the directory  
cat -> to read the text file  

## Explanation  
First, ls was used to find the inhere directory.
Then cd was used to move into the directory.
Finally, cat was used to read the human-readable file and get the password.

## Outcomes  
Learned how to identify and read human-readable files in Linux.
